# Chapter 4 — Environment Setup & Traffic Interception (Elite Notes)

## Essential Tools
- Burp Suite
- FoxyProxy
- SecLists
- ffuf / dirsearch
- Postman / Insomnia

## Burp Suite Modules
- Repeater
- Intruder
- Decoder
- Sequencer
- Proxy
- Logger

## Environment Requirements
- Clean browser profile
- Burp certificate installed
- VM snapshots
- Organized notes

## Interception Workflow
1. Enable proxy
2. Trigger request
3. Intercept & modify
4. Forward or drop
5. Analyze in Repeater
